                                                        Responsive Portfolio Website


## Project Overview
This project is a Responsive Personal Portfolio Website created to showcase my skills, projects, education, and contact details in a professional manner.
This website is fully responsive and works smoothly on desktop, tablet, and mobile devices.

## Problem Statement
Freshers and professionals often face difficulty in presenting their skills and projects professionally online. Traditional resumes are static and do not fully represent practical skills.

## Solution / Approach
* Designed a clean and modern UI
* Implemented responsive layout using CSS
* Added separate sections for skills, projects, contact detail.

## Technologies Used
* HTML - structure of the website
* CSS - styling and responsiveness
* JavaScript - Interactivity
* Media Queries

## Features
* Fully responsive design
* Smooth scrolling navigation
* About Me section
* Skills section
* Project showcase
* Contact sections
* Clean and user-friendly UI

## How to Run the Project
1. Clone the repository
2. open the folder
3. Run index.html in any browser
